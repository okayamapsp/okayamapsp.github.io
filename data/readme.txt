# Data format by column
# 1, starname: name of the star by HD number
# 2, bjd: BJD
# 3, rv: RV
# 4, rverr: error of RV
# 5, fwhm: FWHM of CCF
# 6, bis: bisector invers span (BIS) of CCF
# 7, biserr: error of BIS
# 8, sh: S_H, the index of Ca II H line core emission
# 9, sherr: error of S_H
# 10, qsh: quality of S_H, 1 means good
# 11, tel: the instrument
