# Data format by column
# starname: name of the star by HD number
# bjd: BJD
# rv: RV
# rverr: error of RV
# fwhm: FWHM of CCF
# bis: bisector invers span (BIS) of CCF
# biserr: error of BIS
# sh: S_H, the index of Ca II H line core emission
# sherr: error of S_H
# qsh: quality of S_H, 1 means good
# tel: the instrument
